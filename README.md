## xGame
* A three Console Games
    * Hang Man
    * Ping Pong 
    * Snake

* You can play using console window by pressing button
    * `W` - up
    * `D` - right
    * `A` - left
    * `S` - down

* Built With
    * [C++][cpp-url]
